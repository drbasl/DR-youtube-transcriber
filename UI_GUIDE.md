# UI Guide (Smoke Test Checklist)

## الهدف
تحسين واجهة Streamlit لتكون أجمل وResponsive للهاتف، مع زر نسخ النص كاملًا.

## قبل / بعد (مختصر)
- قبل: توزيع ثابت، زر نسخ غير موجود، ارتفاع النص ثابت، وتجربة هاتفية أقل سلاسة.
- بعد: بطاقات Cards، Toolbar موحد للأزرار، زر نسخ مع Toast، تحسينات Mobile-first.

## Smoke Tests

### 1) اختبار Responsive على الهاتف
- افتح التطبيق من الهاتف أو عبر DevTools (Responsive mode).
- تأكد أن الأعمدة تتحول إلى عمود واحد.
- تأكد أن الأزرار بعرض كامل ولا يوجد overflow أفقي.

### 2) اختبار YouTube captions + نسخ النص
- اختر تبويب YouTube.
- اختر Source = captions.
- فعّل إزالة التوقيت والوسوم (افتراضي ON).
- ابدأ التفريغ.
- اضغط زر "نسخ النص".
- الصق النص في أي مكان وتأكد أنه كامل.
- إذا لم يتم النسخ تلقائيًا: استخدم مربع النسخ اليدوي.

### 3) اختبار رفع MP4 + تنزيل
- ارفع ملف MP4.
- ابدأ التفريغ.
- جرّب تنزيل TXT و JSON وتأكد أنهما يعملان.

## ملاحظات
- زر النسخ يستخدم Clipboard API عند توفره.
- في المتصفحات غير الداعمة يتم عرض مربع نسخ يدوي.
